## 🚀 Features Implemented

- **Sticky Navigation Bar** – Desktop nav with all page links and a Contact Us CTA button
- **Mobile Hamburger Menu** – Slide-in drawer with smooth overlay animation for mobile/tablet
- **Hero Section** – Full-viewport hero with headline, description, checklist, and dual CTA buttons
- **Section 2 (Why Compliance Matters)** – Two-column layout with feature cards (Regulatory Clarity, Hassle-Free Filings, Risk Protection)
- **Pixel-Perfect Design** – Typography, spacing, colors, and layout faithfully replicated from the Figma file
- **Fully Responsive** – Three breakpoints: Desktop (1024px+), Tablet (≤1024px), Mobile (≤500px)
- **Smooth Interactions** – Hover effects on nav links, buttons, and feature cards; smooth scroll behavior

## 📱 Responsive Breakpoints

| Breakpoint | Layout |
|---|---|
| `> 1024px` | Full desktop layout, horizontal nav, two-column sections |
| `≤ 1024px` | Hamburger menu, stacked hero, single-column sections |
| `≤ 500px` | Compact mobile layout, full-width buttons, reduced font sizes |

## 🌐 Deployment

This project is deployed using **[Netlify / Vercel]** *(update as applicable)*.

To deploy your own copy:

2. Drag and drop the project folder onto the Netlify dashboard

1. Go to [vercel.com](https://vercel.com) → Import project from GitHub
2. No build configuration needed — Vercel auto-detects the static site
